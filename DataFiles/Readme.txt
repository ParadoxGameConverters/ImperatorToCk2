This program will convert an Imperator: Rome save game to a Crusader Kings 2 mod.

----
REQUIREMENTS
----
Imperator - Versions 1.3.0 - 2.0.2
CK2       - Version 3.3.4
          - The Old Gods and/or Holy Fury DLC active to play Pagan faith characters
          - The Old Gods DLC active to play Zoroastrian characters
          - Sons of Abraham DLC active to play Jewish characters
          - Rajas of India DLC active to play Eastern faith characters
          - Holy Fury DLC active to have converted bloodlines enabled
Java SE   - Version 16.0.2 or above installed

-----
INSTRUCTIONS
-----
1. Run ConverterFrontend.exe.
2. Fill out all paths and set up any options. Then click Convert.


----
MANUAL METHOD INSTRUCTIONS
----

1. Download the converter and unzip it wherever.
2. Edit configuration.txt to point at your game installs, your savegame location, and to set any options you care about.
3. Run ImperatorToCk2.jar
4. Go into the output folder, then the subfolder named after your save.
5. Copy the folder named after your save and the corresponding mod file into your CK2 mod folder.
6. Start CK2, and select the mod.
